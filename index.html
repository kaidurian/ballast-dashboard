🚂 Generate Trailer

Define one or more trailer trains. Axle #1 of each train sits at the entered X position; every following axle = previous axle's X − line spacing, all sharing the same Y position and VCG. New trains start with the same line spacing and X position as the trailer above them. Split each train's axle range into axle groups (e.g. axles 1–5 @ 25 t, axles 6–10 @ 50 t) and pick one of the 4 fixed groups for each range from the dropdown — or mark a group No line load for axles that carry no cargo weight. Every range assigned to the same group (even across different trains) shares that one color, shown below, in the preview, and on the 2D/3D dashboard. Rows are written to Load_Editor!B (name, e.g. Group_1_line_1) and D:G starting at row 124 (D = Line load t, E = X, F = Y, G = VCG). Use "Load from Excel" to pull an already-generated trailer back in for editing. Tick "Show preview beside input data" or "Show preview in separate window" below to keep the live graphic visible while you edit.
